# Bank Management System (C++)

A menu-driven **Bank Management System** built using **C++** that allows users to create bank accounts, store them permanently using file handling, and display all saved accounts through a console-based interface.

This project focuses on **core C++ concepts**, **robust input validation**, and **safe file handling**.
